# Game Timer Calculator

این یک برنامه ساده برای مدیریت زمان بازی در گیم‌نت‌ها است.

## نحوه استفاده
1. پایتون (Python) را از سایت اصلی دانلود و نصب کنید.
2. فایل `game_timer.py` را دانلود کنید.
3. روی فایل دو بار کلیک کنید تا اجرا شود.
4. با زدن کلید Enter زمان‌گیری را شروع و با زدن دوباره آن، زمان را متوقف کنید.

## ویژگی‌ها
* محاسبه دقیق ساعت و دقیقه
* بسیار سبک و بدون نیاز به نصب هیچ‌گونه نرم‌افزار جانبی
